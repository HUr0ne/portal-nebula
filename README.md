# Portal Nebula

Site institucional estatico para o dominio `portalnebula.com`.

## Estrutura

- `app.js`: servidor Node.js para servir os arquivos estaticos
- `package.json`: scripts e metadados do app Node.js
- `index.html`: conteudo e estrutura da pagina
- `styles.css`: identidade visual e responsividade
- `script.js`: navegacao entre abas
- `CNAME`: dominio customizado usado pelo GitHub Pages

## Publicacao

O projeto pode ser publicado como Web app Node.js na Hostinger.

Configuracao sugerida:

- Framework: Other ou Node.js
- Entry file: `app.js`
- Start command: `npm start`
- Build command: deixe vazio
- Output directory: deixe vazio ou use `.`
